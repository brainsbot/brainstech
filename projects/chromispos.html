Features

Point of sale application for small, medium and large sized businesses. Users are provided with a simple, fast, and efficient way for installing it using its own custom installer that can be used on a number of operating system


  • Windows (7, 8 & 10) | Linux | MacOS
  • Multi sales modes
  • Java 1.8 supported
  • Kitchen Display monitor, support multiple stations
  • Screen resolutions
  • Barcode reader Support
  • Builtin scripting language
  • MySql & PostgreSQL DB supported
  • User security
  • Builtin reports
  • Supports USB printer
  • User customizable Sales screen, Ticket and Receipt layouts
