Overview

This Java SE Fundamentals training teaches students with little or no programming experience to begin to learn programming using the Java programming language. It explores the significance of object-oriented programming, the keywords and constructs of the Java programming language and the steps required to create simple Java technology programs.


Prerequisites

Familiarity with general computer programming concepts and techniques; for more information, see the resources at https://education.oracle.com/.


Objectives

  • Create simple Java classes and describe how to declare a class to access fields and methods in a class
  • Set environment variables to allow the Java compiler and runtime executables to function properly
  • List and describe several key features of the Java technology
  • Describe examples of how Java is used in applications, as well as consumer products
  • Demonstrate Java programming syntax
  • Write a simple Java program that compiles and runs successfully
  • Declare and initialize variables
  • List several primitive data types
  • Use operators, loops, and decision constructs
  • Declare and instantiate Arrays and ArrayLists and be able to iterate through them
  • Describe the benefits of using an Integrated Development Environment (IDE)
  • Use inheritance to declare and define a subclass of an existing superclass
  • Demonstrate polymorphism by implementing a Java interface
  • Encapsulate a class using access modifiers and overloaded constructors
  • Access and create static fields and methods
  • Describe how errors are handled in a Java program
  • Handle a checked exception in a Java application
  • Create Sample Java applications (Hands-On)
